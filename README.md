# To do list

es una aplicación para organizar tareas diarias, permite agregar, marcar como terminadas y eliminar tareas, ademas tiene una alerta de recordatorio para realizar las tareas y el usuario le puede asignar cada cuantos segundos le debe de recordar y muestra una alerta en la pantalla

## Funciones
- agregar tareas
- marcar tareas completadas
- eliminar las tareas
- alerta de recordatorio de tareas

## Tecnologias usadas
- HTML
- CSS
- JavaScript
