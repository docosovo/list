# To do list

es una aplicación para organizar tareas diarias, permite agregar, marcar como terminadas y eliminar tareas, ademas tiene una alerta de recordatorio para realizar las tareas y el usuario le puede asignar cada cuantos segundos le debe de recordar y muestra una alerta en la pantalla la cual viene incluida con un sistema quen permite reproducir un sonido al momento de la alerta ya sea uno de la lista que viene incluida o con la url de la cancion

## Funciones
- agregar tareas
- marcar tareas completadas
- eliminar las tareas
- alerta de recordatorio de tareas con sonido

## Tecnologias usadas
- HTML
- CSS
- JavaScript
